# __PROJECT_TITLE__ 设计说明

## 风格

Studio 现代口播视频主题。口播决定时间线，画面只承载关键词、数字、证据截图、短对比和 CTA 短句。

视觉目标：暖白全画幅画布、上下双层镜像透视格子、黑灰大字、克制蓝色点缀、大留白、细分割线、右下角紧凑口播 PIP、底部无底色黑字字幕。

## 必须满足的布局

- 1920x1080 / 30fps。
- Studio 顶部导航条：高 68px，深色背景，白色进度填充，章节名面向观众。
- 舞台底色：接近 `#f7f8f3`，使用全画幅 `PremiumGridBackground`。
- 背景必须包含上下两层透视格子：顶部像天花板，底部像地面；底部由顶部参数垂直镜像，不能另起一套不同格子。
- 不要把视频内容包进白色圆角卡片、缩小画板、网页 preview 容器或 dashboard shell。
- 右下角圆形口播 PIP，要看到头和肩膀；默认 206px，外圈蓝色描边。
- 底部居中的无底色黑字字幕（底部 96px）；用蓝色加粗强调一两个关键词。
- 主画面只用关键词、列表、数字、对比、截图、录屏作为证据面。

## 设计变量

- 画布：`#f7f8f3`
- 主文字：`#151922`
- 弱文字：`#747982`
- 弱数字：`#b6bbb5`
- 细线：`rgba(28,38,54,0.10)`
- 蓝色强调：`#2f6fff`
- 顶栏：`#202024`
- 字幕文字：`#151922`（无背景底色）
- 字幕强调关键词：`#2f6fff`（加粗）

## 字体

脚手架会从 `public/assets/fonts` 嵌入当前模板字体：

- 中文：`Noto Sans SC`
- 英文/数字：`Space Grotesk`

除非用户明确换主题，不要换成系统字体兜底。

## 模板检查

- 匹配当前 Studio 模板空间系统：内容安全区 `top:196px; left/right:120px; bottom:196px`，68px 顶栏，206px PIP 放在 `right:104px; bottom:150px`，字幕底部 `96px`。
- `PremiumGridBackground` 要实际调用 `PerspectiveGrid`，并渲染上下对称 horizon line、center veil 和 grain。
- 主画面只放关键词。完整口播句子只放字幕。
- 动画由 Remotion 帧驱动，不使用 CSS animation 或 CSS transition。
