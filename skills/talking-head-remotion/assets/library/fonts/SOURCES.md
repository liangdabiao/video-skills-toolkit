# 字体来源

这些字体对应当前 Studio 模板使用的 Google Fonts：

```html
https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@300;400;500;700;900&family=Space+Grotesk:wght@400;500;600;700&display=swap
```

| 本地文件 | 字体族 | 字重 | 来源 |
|---|---|---:|---|
| `NotoSansSC-300.ttf` | Noto Sans SC | 300 | Google Fonts / fonts.gstatic.com |
| `NotoSansSC-400.ttf` | Noto Sans SC | 400 | Google Fonts / fonts.gstatic.com |
| `NotoSansSC-500.ttf` | Noto Sans SC | 500 | Google Fonts / fonts.gstatic.com |
| `NotoSansSC-700.ttf` | Noto Sans SC | 700 | Google Fonts / fonts.gstatic.com |
| `NotoSansSC-900.ttf` | Noto Sans SC | 900 | Google Fonts / fonts.gstatic.com |
| `SpaceGrotesk-400.ttf` | Space Grotesk | 400 | Google Fonts / fonts.gstatic.com |
| `SpaceGrotesk-500.ttf` | Space Grotesk | 500 | Google Fonts / fonts.gstatic.com |
| `SpaceGrotesk-600.ttf` | Space Grotesk | 600 | Google Fonts / fonts.gstatic.com |
| `SpaceGrotesk-700.ttf` | Space Grotesk | 700 | Google Fonts / fonts.gstatic.com |

Google Fonts 字体族使用开放字体授权。复制字体到项目时保留这个文件，方便以后追溯字体来源和当前模板风格。
