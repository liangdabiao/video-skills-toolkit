# 内置音效来源

这些是素材库的可复用音效。脚手架会把它们播种到新项目的 `public/assets/audio/`，再根据最终画面节奏保留真正合适的音效。

| 本地文件 | 来源/授权 | 作用 | 适合什么时候用 | 什么时候不要用 | 备注 |
|---|---|---|---|---|---|
| `sfx-deep-impact.mp3` | Mixkit item 1143, Mixkit License | 重型电影感冲击 | 开头“停”、重大反差、CTA 落点、硬切定格 | 小关键词出现、安静解释段、频繁重复 hit | 主要开场冲击层 |
| `sfx-movie-trailer-epic-impact.mp3` | Mixkit item 2908, Mixkit License | 额外低频冲击层 | 最大的一两个标题 hit | 普通转场、句中强调、音效密集段 | 低音量叠在主冲击下方 |
| `sfx-hard-pop-click.mp3` | Mixkit item 2364, Mixkit License | 锐利瞬态 click | 给冲击加咬字感、快速 UI snap、按钮式落点 | 长讲解段里反复用会疲劳 | 轻轻叠一层即可 |
| `sfx-small-sweep-transition.mp3` | Mixkit item 166, Mixkit License | 柔和转场 sweep | 次级面板进入、小镜头移动、小布局移动 | 需要明显推进感的大场景切换 | 完整 whoosh 太猛时用这个 |
| `sfx-long-pop.mp3` | Mixkit item 2358, Mixkit License | 文字/卡片 pop | 关键词出现、仓库卡片出现、CTA 按钮 pop | 每条字幕都加；严肃数据/证据段 | 默认小揭示音 |
| `sfx-positive-interface-beep.mp3` | Mixkit item 2870, Mixkit License | 确认 beep | 节点出现、进度确认、选中项、成功标记 | 警告、大情绪点、连续快速重复 | 放在人声下面，音量要低 |
| `sfx-whoosh-fast-transition.mp3` | Mixkit item 1492, Mixkit License | 快速 whoosh/riser | 场景切换、章节推进、截图或视频扫入 | 很小的 UI 移动、静态关键词出现、安静解释停留 | 用来制造段落之间的推进感 |
